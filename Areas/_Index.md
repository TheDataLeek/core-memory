# Areas

Ongoing responsibilities — no defined end date. Think: health, finances, a creative practice, professional development.

```dataview
LIST
FROM "Areas"
WHERE file.name = "_Node"
SORT file.mtime DESC
```

---

**To add an area:**
1. Create `Areas/<Name>/`
2. Copy `Templates/Area-Node.md` → `Areas/<Name>/_Node.md`
3. Copy `Templates/Area-Todo.md` → `Areas/<Name>/_Todo.md`
